An extensible Markdown to HTML converter written in Javascript

View on GitHub Download .zip Download .tar.gz

Showdown is a Javascript Markdown to HTML converter, based on the original works by John Gruber. Showdown can be used client side (in the browser) or server side (with NodeJs). Check a live demo here

Get it...

  • Download tarball: download the latest release tarball directly from releases
  • CDN:
    • rawgit: https://cdn.rawgit.com/showdownjs/showdown/<version tag>/dist/showdown.min.js
    • cdnjs: https://cdnjs.cloudflare.com/ajax/libs/showdown/<version tag>/showdown.min.js
  • Bower: bower install showdown
  • npm: npm install showdown
  • NuGet: PM> Install-Package showdownjs - NuGet package here

Use it...

//node
var showdown = require('showdown');
var converter = new showdown.Converter();
var htmlOutput = converter.makeHtml('*Hello* __World__');

Check our wiki pages for a more in-depth documentation, such as syntax support, examples, extensions, etc...
